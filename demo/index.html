Design Tokens Demo

New

This demo showcases multi-brand theming using component-level design tokens. Each brand has a unique personality expressed through different border radii, fonts, and component styling - not just colors!

Try switching brands: Watch buttons morph from pill-shaped (Brand A: 100px) to sharp corners (Brand B: 0px). Notice how fonts change from Satoshi to Zodiak to IBM Plex. All themes are loaded once, and switching happens instantly via CSS class changes on <html>.
